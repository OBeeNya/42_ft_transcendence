Un utilisateur essaye de rejoindre un channel :

- l'utilisateur n'est pas membre du channel et ce dernier est public --> il le rejoint

- le channel est privé --> l'utilisateur a besoin d'une invitation 
  ET/OU le propriétaire du channel doit approuver la demande pour rejoindre

- le channel est protégé par mot de passe : l'utilisateur doit fournir le bon mdp pour rejoindre

- l'utilisateur est deja membre du channel

- Le channel est plein

- L'utilisateur a ete bloque par le propriétaire du channel ou
  par l'administrateur

- L'utilisateur a été banni du channel

-----------------------------------------------------------------------------------------
Problèmes :

1) Pas de statut "In game", et le mode spectateur ne marche pas dans l'interface amis.

----------------------------------------------------------------------------------------- 

Lorsque l'utilisateur clique sur l'icone de l'oeil, verifier si playing est true.
Si true et que deja 2 joueurs--> rediriger vers /pongGame

const spectating = async () =>
{
	try
	{
		const players = await ax.get('pong/getPlayers', {headers: {Authorization: `Bearer ${token}`}});

			if (players.data.length !== 2)
			{
				const messageSpectating = document.getElementById("messageSpectating");

				if (messageSpectating)
					messageSpectating.textContent = "No game is currently being played";
			}

			else
				navigate('/pongGame');

		}
		catch
		{
			console.error('could not spectate');
		}
}
